# July Tech Roundup: Hot Off the Press

What time is it? Tech roundup time! Here are the latest updates, tech tips, and transition guides, just for you. 

**SP4 Release**

We have released our fourth Service Pack (SP4) for the Vectorworks 2018 product line. This Service Pack includes multiple fixes and improvements in the following areas: 

* Significant overall application stability improvements
* Corrected issues related to user origin shifts, roof plans, and document settings
* Several quality improvements for Marionette workflows
* Multiple improvements to the performance and stability of project sharing
* Several IFC import and export improvements

This Service Pack is available for all licenses as a downloadable updater. To install the Service Pack, please [click here ](http://www.vectorworks.net/support/downloads/product-updates?utm%5Fcampaign=blog&utm%5Fsource=intext&utm%5Fmedium=planetvectorworks&utm%5Fcontent=techroundup0718)or select "Check for Updates" from the Vectorworks menu (Mac) or Help menu (Windows).

Need help? Contact us at [tech@vectorworks.net](../../../net/vectorworks/blog/index.html).
