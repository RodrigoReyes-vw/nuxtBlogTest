## iMac Pro For All

We now support Vectorworks 2018 running on all iMac Pro desktops. Shortly after the launch of the iMac Pro, we discovered an instability when running tests in Vectorworks 2017 and 2018\. Apple has since released a patch in macOS 10.13.5 that corrects this issue.
